    Checkliste

    Markdown gibt Ihnen zusätzlich die Möglichkeit, Checklisten zu erstellen. Diese erscheinen mit einem Kästchen, das per Knopfdruck aktiviert werden kann. Sie können auch bereits beim Erstellen der Liste Häkchen setzen. Nutzen Sie hierfür eckige Klammern und ein X. Zwischen den eckigen Klammern ein Leerzeichen.

    - [ ] A
- [x] B
- [ ] C
    • [ ] A
    • [x] B
    • [ ] C

    Checkliste in Tabellenform

    | Aufgabe | Status |
|----------|--------|
| A | [x] |
| B | [ ] |
| C | [ ] |
    Aufgabe Status
    A [x]
    B [ ]
    C [ ]

    Darstellung von Code

    • Apostroph vor und nach dem Wort siehe code.
    • Zwei Apostrophe an Anfang und Ende hebt jeweils ein Apostroph auf, siehe `code`.
    • Drei Apostrophe Anfang und Ende, sowie der Name der Programmiersprache. Damit wird highlighting aktiviert
